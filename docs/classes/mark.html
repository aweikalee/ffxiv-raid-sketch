  • setAlias(name: IMarkAlias, alias: string): void
    • Defined in Mark.ts:103

    为目标标记设别名

    比如 攻击1 和 attck1 分别是中英文的官方名称

    我可以通过 Mark.setAlias('攻击1', '攻1') 设置别名

    之后则可以使用 new Mark('攻1') 获得同样的目标标记

    Parameters

    • name: IMarkAlias

      中英文官方名称 / 已设置成功的别名

      别名
